# Semantic Resume Web Page

## Overview
A clean, responsive **resume / portfolio web page** built using **semantic HTML5** best practices.
This project focuses on document structure without relying on frameworks.

This project contains multiple sections which a resume should have with details of each sections following the rules of sementic HTML and best practices.

- Semantic HTML5 structure
- Clean and readable layout
- Easy to customize

## What are the sections?

- Header
- About Me
- Skills
- Experience
- Projects
- Education
- Contact
